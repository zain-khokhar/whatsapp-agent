## Getting Started

1. **Clone the repository**
     ```bash
     git clone https://github.com/yourusername/whatsapp-agent.git
     cd whatsapp-agent
     ```

2. **Install dependencies**
     ```bash
     npm install
     ```

3. **Configure your environment**  
     Edit `.env` with your WhatsApp credentials and settings.

4. **Run the agent**
     ```bash
     npm start
     ```
